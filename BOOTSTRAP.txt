class name 

colors
    1. primary -> blue
    2. secondary -> gray
    3. success -> green
    4. warning -> yellow
    5. danger -> red
    6. info -> skyblue
    7. light -> white 
    8. dark -> black

bg -> background
text -> content 

bg-[colors]
text-[colors]

text allignment
    1. text-center -> alignment in center horizontally
    2. text-start -> alignment in left 
    3. text-end -> alignment in right

container -> provide the left margin and right margin 
    1. container-fluid -> occupies the entire width

    breakpoints - xs, sm, md, lg, xl, xxl
    2. container-[breakpoints]

Margins 
    num=0,1,2,3,4,5, auto
    1. mt-[num] -> margin top
    2. mb-[num] -> margin bottom
    3. ms-[num] -> margin left
    4. me-[num] -> margin right
    5. mx-[num] -> margin left and right
    6. my-[num] -> margin top and bottom
    7. m-[num] -> margin on all sides

Padding
    num=0,1,2,3,4,5, auto
    1. pt-[num] -> padding  top
    2. pb-[num] -> padding  bottom
    3. ps-[num] -> padding  left
    4. pe-[num] -> padding  right
    5. px-[num] -> padding  left and right
    6. py-[num] -> padding  top and bottom
    7. p-[num] -> padding on all sides


navigation bar
    1. navbar
    2. navbar-brand (comes to the left of the navbar)
    3. nav (get output inline)
    4. nav-link (navigation links, removes undeline from links)
